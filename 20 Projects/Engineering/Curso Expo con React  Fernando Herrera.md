# Project: Curso Expo con React

## Overview
> **Goal:** Aprender sobre la creación de aplicaciones móviles usando Expo + React Native. Enfocado en prototipado rápido y despliegue en Android/iOS.

- **Why:** Una guía práctica para construir, testear y publicar una app móvil con Expo y React Native.
- **Success Criteria:**
  - Al subir mi primera aplicación a las plataformas
  - Al tener descargas fuera de mi circulo social

## Objectives & OKRs
- [ ] Crear mi primera aplicación
- [ ] Subirla a Play Store, Google Play
- [ ] Monetizar la aplicación

## Milestones
- [ ] **Phase 1:** Implementar plantilla inicial con Expo CLI
- [ ] **Phase 2:** Completar secciones: Tipos de Navegación (Stack, Tab, Drawer)
- [ ] **Phase 3:** Crear componente de ejemplo y aplicar Native Wind
- [ ] **Phase 4:** Subir primera build a Google Play (beta)
